üü© Truthy and Falsy Values
JavaScript treats some values as "falsy" ‚Äî meaning they evaluate to false in conditionals. Everything else is "truthy".

‚ùå Falsy Values:

"" (empty string)

null

undefined

0

‚úÖ Truthy Values:
Any non-zero number (e.g. 1, -1)

Any non-empty string (e.g. "abc")

Objects, arrays, etc.
